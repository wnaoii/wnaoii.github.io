CSS盒子模型

div标记

  • div是 division 的缩写，意为“分割，区域”。
  • 其中可以嵌套大多数 HTML 标记，也可以嵌套多个 div 标记。
  • 外边距（Margin）
  • 边框（Border）
  • 内边框（Padding）

盒子的宽与高

  • width 控制宽度 height 控制高度

  • 宽度（width）和高度（height）属性仅适用于块级元素，对行内元素无效。

边框属性

  • solid: 单实线
  • dashed: 虚线
  • dotted: 点线
  • double: 双实线

border-style: 上边 右边 下边 左边；

边距属性

 1. 内边距
     - padding-top: 上内边距
     - padding-right: 右内边距
     - padding-bottom: 下内边距
     - padding-left: 左内边距
     - padding: 左内边距 右内边距 上内边距 下内边距 , 四个方向内边距相同
  1. 外边距
    • margin-top: 上外边距
    • margin-right: 右外边距
    • margin-bottom: 下外边距
    • margin-left: 左外边距
    • margin: 左外边距 右外边距 上外边距 下外边距 , 四个方向外边距相同
