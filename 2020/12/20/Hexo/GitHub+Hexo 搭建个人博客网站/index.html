GitHub+Hexo 搭建个人博客网站

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。

Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，

即可利用靓丽的主题生成静态网页。

准备事项

Node.js

  • Node.js下载地址

  • 测试是否安装成功node -v显示版本号即成功

  • npm包管理器

    • Hexo框架需要通过npm管理工具进行安装

    • 安装Node.js时默认安装npm包管理器

    • 测试是否安装成功npm -v显示版本号即成功

    • npm换源

      • 以下两种任选一种即可

      1. 第一种：

        下载cnpm切换到taobao源

        npm install -g cnpm --registry=https://registry.npm.taobao.org

      2. 第二种：

        安装nrm

        • npm的镜像源管理工具

        • 通过npm下载nrm

Git

  • Git下载地址
  • 测试是否安装成功git -v显示版本号即成功
